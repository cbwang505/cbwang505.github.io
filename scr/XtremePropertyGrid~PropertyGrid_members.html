PropertyGrid Overview

Xtreme Property Grid ActiveX Control v15.3

PropertyGrid Control Members

Public Methods

 AboutBoxDisplays a dialog box that specifies the current version of the control.
 AddCategoryAdds a new category to the Property Grid control.
 BeginUpdateThe BeginUpdate method will  "lock" the property grid.  This should be called when adding, deleting, or modifying a large number of items.
 EditItemPlaces an item into edit mode via code.
 EndUpdateThe EndUpdate method is called to end the update process started by BeginUpdate.  EndUpdate will "unlock" the grid after you have finished adding, deleting, or modifying a large amount of items.
 FindItemFinds and returns a PropertyGridItem, searching by the Caption or Tag value.
 HitTestTests a point to see if a PropertyGridItem is present.  If there is, then a reference to that item is returned.
 InsertCategoryThe InsertCategory method is used to insert a new category to the Property Grid at a specific position.

Public Properties

 AllowEditSpecifies whether items in the PropertyGrid are allowed to be edited by the user.  This can be thought of as Read-Only mode when set to False.
 BorderStyleGets\sets the border style for Property Grid.
 CategoriesGets the categories collection of this PropertyGrid.
 EditHwndHandle of Edit Window where user enters text in edit mode.
 EditModeSpecifies whether the PropertyGrid is currently in edit mode (An item is being edited).
 EnabledReturns/sets a value that determines whether an object can respond to user-generated events.
 EnableMarkupAllows the use of a XAML snippet to be used for the Title, Description and Tooltip of items in the property grid.
 FontGets/sets the font used to display text in the property grid.
 HelpBackColorGets/sets the background color for the PropertyGrid help panel.
 HelpForeColorGets/sets the forecolor\text color for the PropertyGrid help panel. 
 HelpHeightGets/Sets the height of the built in help box at the bottom of the PropertyGrid.
 HelpVisible

Gets/Sets a Boolean value indicating whether the built in help panel is visible.

 HideSelectionSpecifies whether the selection in a Property Grid is hidden when the control loses focus.
 HighlightChangedButtonItemsGets\sets whether the text in the in-place button will be bold if HighlightChanged is used.
 HighlightChangedItemsSpecifies whether item's who's value has changed from the DefaultValue will be displayed in bold.
 hWndReturns the handle to the PropertyGrid control.
 IconsA collection of icons that can be used in the PropertyGridItem and PropertyGridItemConstraint objects.
 LineColorGets/sets the line color for the property grid.  This is the area of the property grid that surrounds the property grid items.
 LockRedrawReturns whether the grid will be redrawn after an item is added.
 NavigateCaptionsSpecifies whether item captions will receive focus in the grid when items are navigated when pressing the Tab key.
 NavigateItemsThe NavigateItems property is used to specify how the items in the grid are navigated when pressing the Tab key.
 PaintManagerPointer to the paint manager used to draw the property grid.
 PropertySortGets/Sets the way in which PropertyGridItems are sorted in the PropertyGrid.
 RightToLeftGets or sets a value indicating whether the Property Grid elements are aligned to support locales using right-to-left fonts.
 SelectedReturns a reference to the PropertyGridItem that is currently selected\has focus.
 ShowInplaceButtonsAlwaysSpecifies whether to always display all in-places buttons for all items regardless of which item currently selected.
 SplitterPosGets/Sets the position of the splitter that divides the item description from the item data.
 ToolBarVisible

Gets/Sets a Boolean value indicating whether the built in toolbar is visible.

 ToolTipContextThe ToolTipContext object represents the tool tip window that is used to display tool tips.
 TopIndexReturns the index of the topmost item displayed in the Property Grid.
 VariableHelpHeightGets\sets whether the user can resize the help box with the mouse.
 VariableItemsHeightGets\Set whether variable item height is allowed in the property grid.
 VariableSplitterPosGets\sets whether the user can move the property grid splitter.
 VerbsThe Verbs property is a collection of PropertyGridVerb objects that are displayed in the verb panel.
 VerbsHeightThe VerbsHeight property is used to get or set the height of the verb panel when visible.
 ViewBackColorGets/Sets the view backcolor for the PropertyGrid.  This is the background color of the PropertyGridItems.
 ViewCategoryForeColorGets/Sets the color used to display the text of Category PropertygridItems.
 ViewForeColorGets/Sets the view forecolor for the property grid.  This is the foreground color of the property grid items.
 ViewReadOnlyForeColorThe ViewReadOnlyForeColor is used to get or set the text color of read only items.
 VisualThemeGets/Sets the visual theme that is used by PropertyGrid

Events

 AfterEditOccurs after an edit operation is complete.
 CancelEditOccurs if the edit operation has been canceled.
 DblClickOccurs when the description half (left side) of a PropertyGridItem has been double clicked, and when a category item is double-clicked.  The item that was double-clicked is passed to this event.
 EditChangedOccurs when the contents of an item has changed while in edit mode.
 ExpandChangedOccurs when a expand\collapse button is clicked.
 InplaceButtonDownOccurs when a combo button or expand button that was added by the developer is clicked.  Built in buttons such as the color picker button do not trigger this event. It is used only for custom buttons.
 KeyDownOccurs when a key is pressed down and the property grid has focus.  This event will not be triggered while editing a property grid item, only when an item or category is selected.
 KeyUpOccurs when a key is released and the property grid has focus.  This event will not be triggered while editing a property grid item, only when an item or category is selected.
 MouseDownOccurs when the user presses a mouse button.
 MouseMoveOccurs when the user moves the mouse.
 MouseUpOccurs when the user releases a mouse button.
 PropertySortChangedOccurs when the PropertySort property has changed.
 RClickOccurs when you right-click anywhere on the property grid.  The item that was right-clicked is passed to this event.  This event is not triggered if you right-click the item you are currently editing.
 RequestEditOccurs when an editable item goes into edit mode.
 SelectionChangedOccurs when a new item is selected in the PropertyGrid.
 ValueChanged

Occurs when the value of a PropertyGridItem has changed.

 VerbClickThe VerbClick event occurs when a user clicks on a verb or when a verb has focus and the enter key is pressed.
