Xtreme Calendar ActiveX Control v15.3

SetColor Method

Description

The SetColor function sets the colors for one or more display elements. Display elements are the various parts of a window and the display that appear on the system display screen.

Display elements to be changed.  The color of the selected element will be replaced with the color value in the Color parameter.
Color
The new red, green, blue (RGB) color value for the display element specified by Index.

Remarks

Using SetColor for the Calendar control will change the color of the selected element of the Calendar to the new specified color.  This will not effect the color for any other control.

 

SetColor applies colors globally, this means that all Calendar controls will uses these colors.
