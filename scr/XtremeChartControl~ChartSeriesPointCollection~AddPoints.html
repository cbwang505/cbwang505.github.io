Xtreme Chart Pro ActiveX Control v15.3

AddPoints Method

Description

Adds a batch of points.  (Use to allocate memory for a very large number of points)

Syntax

Public Sub AddPoints( _
   ByVal Count As Long _
)

Parameters

Count
Number of points to add.

Remarks

AddPoints is a way to optimize adding a large number of points. Memory will be allocated once for the specified number of points. This will dramatically increase the speed at which the points are added when adding a large number of points.  You can both AddPoints using Addpoints and any other "Add" method for the same series.

You will have to manually add values to the points after they have been added. Below you will find a few samples.

Example

'********************************************************

'Area chart with 1 series added using AddPoints. 

'********************************************************

        
    ChartControl.Content.Legend.Visible = True
    ChartControl.Content.Legend.HorizontalAlignment = xtpChartLegendNearOutside
