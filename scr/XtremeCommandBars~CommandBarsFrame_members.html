Events

 CommandBarKeyDownOccurs when a key is pressed and a popup menu is open, a popup menu has focus, or a menu item in a popup menu has focus.

 CustomizationResetAll 
 CustomizationResetToolBarOccurs when the "reset" button is used to reset a toolbar back to the default layout.
 ExecuteOccurs when a command bar control is clicked.
 GetActiveDocumentIDOccurs when a MDI document becomes active, or all documents are closed.  It used to decide which MDI menus to display based on the type of document that is active.
 GetClientBordersWidthOccurs when the area occupied by controls docked to the edges of the form has to be retrieved.
 InitCommandsPopupOccurs when a popup control is about to be expanded/shown.
 OLECompleteDragOccurs when the commandbar's component is dropped onto a target component, informing the commandbar's component that a drag action was either performed or canceled.
 OLEStartDragOccurs when the commandbar's OLEDrag method is performed, or when the commandbar component initiates an OLE drag/drop operation when the OLEDragMode property is set to Automatic.

This event specifies the data formats and drop effects that the source component supports. It can also be used to insert data into the DataObject object.

 RepositionAlignedControlsOccurs when controls are to be repositioned on a MDI form.
 ResizeOccurs when the command bar layout changes.
 ResizeClientOccurs when the command bar layout changes.
 SpecialColorChangedOccurs when a special color has been changed in the command bar
 ToolBarContextMenuOccurs when the CommandBars default context menu is displayed.
 ToolBarVisibleChangedOccurs when the visible status of a Toolbar changes.
 TrackingModeChangedOccurs when the command bars tracking mode has changed.  Tracking mode is similar to get\lost focus for a command bar.
 UpdateOccurs when a command bar control has been updated.
