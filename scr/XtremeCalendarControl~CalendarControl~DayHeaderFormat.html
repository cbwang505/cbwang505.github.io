DayHeaderFormat Property

CalendarControl Control  | FullDateMode Property

Xtreme Calendar ActiveX Control v15.3

DayHeaderFormat Property

Description

Day header format to use when FullDateMode is True.

Property type

Read-write property

Syntax (Visual Basic)

Public Property DayHeaderFormat() As String

Remarks

FullDateMode tells the calendar whether to use the format sting stored in DayHeaderFormat to format the day view day headers.  When set to False, the calendar will use the system settings to display the header information.  This format only applied to views where the date it completely displayed like day view and work week view.

 

Note: Nothing will be displayed in the day view day headers if you set FullDateMode = True and leave DayHeaderFormat blank.

 

Need to call on load:
CalendarCtrl.FullDateMode = True
CalendarCtrl.DayView.DayHeaderFormat = "%a, %b %d, %y"

Valid Format Strings:
a - Abbreviated weekday name
A - Full weekday name
b - Abbreviated month name
B - Full month name
c - Date and time representation appropriate for locale
d - Day of month as decimal number (01 &#8211; 31)
H - Hour in 24-hour format (00 &#8211; 23)
I - Hour in 12-hour format (01 &#8211; 12)
j - Day of year as decimal number (001 &#8211; 366)
m - Month as decimal number (01 &#8211; 12)
M - Minute as decimal number (00 &#8211; 59)
p - Current locale's A.M./P.M. indicator for 12-hour clock
S - Second as decimal number (00 &#8211; 59)
U - Week of year as decimal number, with Sunday as first day of week (00 &#8211; 53)
w - Weekday as decimal number (0 &#8211; 6; Sunday is 0)
W - Week of year as decimal number, with Monday as first day of week (00 &#8211; 53)
x - Date representation for current locale
y - Year without century, as decimal number (00 &#8211; 99)
Y - Year with century, as decimal number
z, %Z - Either the time-zone name or time zone abbreviation, depending on registry settings; no characters if time zone is unknown

You can even add some special text in any place of format like this:


CalendarCtrl.DayView.DayHeaderFormat = "Office %a, %b %d, %y Schedule"
