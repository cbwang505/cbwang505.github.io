Resizer Overview

Xtreme Suite Controls ActiveX Control v15.3

Resizer Control Members

Public Methods

 AboutBoxDisplays a dialog box that specifies the current version of the control
 AddControlAdds a control to be handled by the resizer.  Controls handled by the resizer can get resized automatically when the resizer is resized.

Public Properties

 AutoSizeGets or sets whether the resizer control will auto-size to the size of it's container.
 BackColorGets or sets the background color of the control.
 BorderStyleGet or sets the border style of the resizer control.
 ClientMaxHeightGets or set the maximum height of the resizer control.
 ClientMaxWidthGets or set the maximum width of the resizer control.
 ClientMinHeightGets or set the minimum height of the resizer control.
 ClientMinWidthGets or set the minimum width of the resizer control.
 HScrollLargeChangeGets or sets the amount the resizer is scrolled horizontally when the scroll bar is clicked above or below the gripper (not on the scroll buttons).
 HScrollMaximumGets or sets the maximum size the contents of the resizer can be scrolled horizontally.
 HScrollPositionGets or sets the position of the horizontal scroll bar gripper.
 HScrollSmallChangeGets or sets the amount the resizer is scrolled horizontally when the scroll bar buttons are clicked.
 hWndGets the window handle that the control is bound to.
 MouseIconGets or sets the cursor that is displayed when the mouse pointer is over the control.
 ShowSizeIconGets or sets whether a resizer icon will be displayed in the bottom right corner of the resizer control.
 VScrollLargeChangeGets or sets the amount the resizer is scrolled vertically when the scroll bar is clicked above or below the gripper (not on the scroll buttons).
 VScrollMaximumGets or sets the maximum size the contents of the resizer can be scrolled vertically.
 VScrollPositionGets or sets the position of the vertical scroll bar gripper.
 VScrollSmallChangeGets or sets the amount the resizer is scrolled vertically when the scroll bar buttons are clicked.

Events

 DblClickOccurs when the user presses and releases a mouse button and then presses and releases it again over the resizer control.
 HScrollOccurs when the user clicks the horizontal scroll bar of the control.
 MouseDownOccurs when the user presses a mouse button.
 MouseMoveOccurs when the user moves the mouse.
 MouseUpOccurs when the user releases a mouse button.
 MouseWheelOccurs when the mouse wheel moves while the resizer or controls inside the resizer control has focus.
 VScrollOccurs when the user clicks the vertical scroll bar of the control.
