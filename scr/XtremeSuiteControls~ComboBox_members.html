Public Methods

 AboutBoxDisplays a dialog box that specifies the current version of the control
 AddItemAdds an item to the combobox control.

 SortedGets or sets a value indicating whether the items in the combo box are sorted.

Events

 ChangeIndicates the contents of the combobox have changed.
 ClickOccurs when the user presses and then releases a mouse button over the combobox. It can also occur when the value of the combobox is changed.
 DblClickc
 DropDownOccurs when the drop-down portion of a ComboBox is shown.
 KeyDownOccur when the user presses a key while a control has the focus. (To determine ANSI characters, use the KeyPress event.)
 KeyPressOccurs when the user presses and releases an ANSI key.
 KeyUpOccur when the user releases a key while a control has the focus. (To determine ANSI characters, use the KeyPress event.)
