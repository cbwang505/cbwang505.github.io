Xtreme Property Grid ActiveX Control v15.3

InsertChildItem Method

Description

The InsertChildItem method is used to insert a new PropertyGridItem at a specific position within a collection of items.

Syntax

Public Function InsertChildItem( _
   ByVal Index As Integer, _
   ByVal Type As PropertyItemType, _
   ByVal Title As String, _
   ByVal Value As Variant _
) As PropertyGridItem

Parameters

Index

The position in the collection of items to insert the new item.

Type
ValueDescription
PropertyItemStringSpecifies a PropertyGridItem of type String.
PropertyItemNumberSpecifies a PropertyGridItem of type Number.
PropertyItemBoolSpecifies a PropertyGridItem of type Boolean.
PropertyItemColorSpecifies a PropertyGridItem of type Color.
PropertyItemFontSpecifies a PropertyGridItem of type PropertyGridItemFont.
PropertyItemDoubleSpecifies a PropertyGridItem of type Double.
PropertyItemDateSpecifies a PropertyGridItem of type PropertyGridItemDate.
PropertyItemPictureSpecifies a PropertyGridItem of type Picture.
PropertyItemEnumSpecifies a PropertyGridItem of type Enum.
PropertyItemEnumFlagsSpecifies a PropertyGridItem of type EnumFlags.  This will create a Flags value item in the Property Grid control.
PropertyItemCategorySpecifies the PropertyGridItem is a Category.
PropertyItemMultilineStringSpecifies a PropertyGridItem that contains a multi line string.  When using this property a dropdown button will appear next to the item text that when clicked will display a edit window with the entire line of text.
PropertyItemOptionSpecifies a PropertyGridItem of type PropertyGridItemOption.

Type of item to insert.

Title

The caption/title to give the item.

Value

The value to assign to the item.

Return Type

Pointer to the item at the specified index that was inserted into the collection of items.

Remarks

The InsertChildItem method creates a new PropertyGridItem and inserts it at the specified Index in the collection of items.  Items can be added to the Categories collection of items or to Child Items collection of items as in the picture below.  Items added to the Categories collection of items will be used categories.  Items added to a category's collection of child items will be displayed in the category that they are added to.
