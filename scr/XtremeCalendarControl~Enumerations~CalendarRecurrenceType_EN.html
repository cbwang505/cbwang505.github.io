CalendarRecurrenceType Enumeration

Description

Types of Recurrence Pattern Types.

Members

MemberDescription
xtpCalendarRecurrenceUnknownUnknown Recurrence Type.
xtpCalendarRecurrenceDailyDaily Recurrence.  Recur every N days.  N = { 1, 2 .. }.
xtpCalendarRecurrenceWeeklyWeekly Recurrence.  Recur every N weeks on the specified 'DaysMask'.  N = { 1, 2 .. }.
xtpCalendarRecurrenceMonthlyMonthly Recurrence.  Recur every D day of every N months.  D = { 1, 2 .. 31},  N = { 1, 2 .. }.
xtpCalendarRecurrenceMonthNthMonthly Recurrence (Recurs on the same day during a specific week, i.e, Recurs on the 3rd Monday of month(M) every N months).  Recur the 1st, 2nd, 3rd, 4th, or Last X of every N months.  X = XTPCalendarWeekDay, N = { 1, 2 .. }.

 

See Nth Notes section below.

xtpCalendarRecurrenceYearlyYearly Recurrence.  Recur every day(D) of month(M) each year.  M = { 1, 2 .. 12}, D = { 1, 2 .. 31}.
xtpCalendarRecurrenceYearNthYearly Recurrence (Recurs on the same day during a specific week, i.e, Recurs on the 3rd Monday of Month(M) every year).  Recur the 1st, 2nd, 3rd, 4th, or Last X of month(M).  X = XTPCalendarWeekDay, M = { 1, 2 .. 12}.

 

See Nth Notes section below.

Remarks

The RecurrenceType property of the Recurrence Pattern Options uses the CalendarRecurrenceType Enumeration to specify the type of an event.

 

Nth Notes:

Both the MonthlyNth and YearlyNth recurrences use a Nth value that indicates if the event should occur on the 1st, 2nd, 3rd, 4th or Last XTPCalendarWeekDay of a month.

When specifying the Nth part of the MonthlyNth and YearlyNth recurrences, the following values should be used:

Nth Value
First 1
Second 2
Third 3
Fourth 4
Last 5

 

The MonthlyNth value is set using the MonthlyNthWhichDay property.

The YearlyNth value is set using the YearlyNthWhichDay property.
