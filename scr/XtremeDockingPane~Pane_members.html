Pane Overview

Xtreme Docking Pane ActiveX Control v15.3

Pane Object Members

Public Methods

 AttachToAttaches a pane to another pane.  This will group the panes together placing one pane inside the other creating a nested group of docking panes.
 CloseCloses the pane.
 GetClientRectRetrieves the dimensions of the client rectangle, that is, the area on the form which does not have any docking panes.
 HideHides the pane.
 SelectSelects/sets focus to the docking pane. This makes the pane active.
 SetHandleSets the handle of the child window displayed in the pane.

Public Properties

 ClosedGets/ Sets a Boolean value indicating that the docking pane is open/closed.
 EnabledSpecifies the extent that a pane is enabled\disabled.
 FloatingGets a Boolean value indicating that the docking pane is floating/docked.
 HandleGets/Sets the handle of the child window displayed in the docking pane.
 HiddenGets\sets a Boolean value indicating that the docking pane is hidden/visible.
 IconIdGets/sets the numeric identifier of the icon used by the docking pane.  This will be set to the docking pane Id by default if this property is not manually set.  This is the number used to associate a docking pane with an icon in an ImageList control.
 IdGets/Sets the numeric identifier of the docking pane.  This is used to identify the pane and is used in operations such as finding a pane, determining which pane was clicked, hiding a pane, etc...
 MaximizedGets or sets whether the pane is maximized when the Maximize\Restore Buttons are displayed on the pane caption bar for panes that are adjacent to each other.  i.e. when the panes use the "Cascading" functionality seen in Visual Studio.
 MaxTrackSizeUsed to set the maximum size that a Pane can be re-sized to.
 MinTrackSizeUsed to set the minimum size that a Pane can be re-sized to.
 OptionsGets/ Sets the options of the pane
 ParentContainerReturns a PaneContainer object that points to the parent container for this pane.
 PositionReturns the position of the Pane relative to the Client area.
 SelectedGets/sets a Boolean value indicating that the pane is currently selected/not selected.  Setting this property to true has the same effect as calling the Select method.
 TabCaptionGets\set the Pane's tab button caption.
 TabColorGets\sets the color of the tab button for the pane.
 TagGets/Sets the tag of the pane.  Stores any extra data needed for your program.
 TitleGets\sets the title of the docking pane.  This title will be displayed in the title bar of the docking pane.
 TitleToolTipGets\sets the tooltip displayed to the Pane's caption.
 TypeReturns the type of this pane.
