CreateFromXML Method

TaskDialog Control  | ShowDialog Method

XMLString
The contents of a XML file.  This string must contain the entire contents of the XML file.
Section
Name of the XML resource to load.  This should be the section that contains the dialog you want to load.
Parameters

Values to use in the text descriptions of the dialog.  More than one paramater can be passed in and can be stored in a string array.  These values will be inserted into any text portion of the dialog that contains a followd by an number.  The number corresponds to a parameter in the array and starts with 1.

 

For example, 3 paramaters are used in the content section of this dialog1\nType:  2\nFrom3</Content>

 

The code to setup the parameters would look like this:

    Dim Params(3) As String
    Params(0) = "SampleFileName.txt"
    Params(1) = "Text Document"
    Params(2) = "www.Codejock.com"

    taskDialog.CreateFromXML LoadDialogs, "Dialogs/Dialog[@Name='Dialog007']", Params

 

Or you can do it like this:


    taskDialog.CreateFromXML LoadDialogs, "Dialogs/Dialog[@Name='Dialog007']", Array("SampleFileName.txt", "Text Document", "www.Codejock.com")

CreateFromXML Method

Description

Creates a dialog from a XML resource file.

Syntax

Public Function CreateFromXML( _
   ByVal XMLString As String, _
   ByVal Section As String, _
   Optional ByVal Parameters As Variant _
) As Boolean

Parameters

XMLString
The contents of a XML file.  This string must contain the entire contents of the XML file.
Section
Name of the XML resource to load.  This should be the section that contains the dialog you want to load.
Parameters

Values to use in the text descriptions of the dialog.  More than one paramater can be passed in and can be stored in a string array.  These values will be inserted into any text portion of the dialog that contains a followd by an number.  The number corresponds to a parameter in the array and starts with 1.

 

For example, 3 paramaters are used in the content section of this dialog1\nType:  2\nFrom3</Content>

 

The code to setup the parameters would look like this:

    Dim Params(3) As String

Remarks

You can use a utility such as TDPad (http://shellrevealed.com/files/folders/code/tags/TDPad/default.aspx) to design and create task dialogs that are stored in a XML file that can be loaded at any time.  For example, the dialog below was created using TDPad:

Example

[Example XML Dialog ()] Below is the XML to create a dialog that will except 3 parameters.

<?xml version="1.0" encoding="UTF-8"?> 
<Dialogs xmlns="http://www.codejock.com/schema/TDSchema.xsd"> 
 <Dialog Name="Dialog007" AllowCancel="true" EnableHyperlinks="true" MessageBoxStyle="true"> 
  <WindowTitle>File Download</WindowTitle> 
  <MainInstruction Image="*Warning">Do you want To Open %3 Or save this file?</MainInstruction> 
  <Content>Name:  %1\nType:  %2\nFrom:  %3</Content> 
  <Buttons Default="2"> 
   <Cancel/> 
   <Button ID="100">&Open</Button> 
   <Button ID="101">&Save</Button> 
  </Buttons> 
  <Verification Checked="true">&Always ask before Open</Verification> 
  <Footer Image="*Shield">While files from Internet can be useful, some files can potentially harm your computer. If you Do Not trust the source, Do Not Open Or save this file. <a href="http://www.microsoft.com">What's the risk?</a></Footer> 
 </Dialog> 
</Dialogs> 
 
 
 

[Load From XML File (Visual Basic)] This code will load and display the task dialog in the XML code above.

taskDialog.Reset
taskDialog.CommonButtons = 0

Dim Params(3) As String
Params(0) = "SampleFileName.txt"
Params(1) = "Text Document"
Params(2) = "www.Codejock.com"

taskDialog.CreateFromXML LoadDialogs, "Dialogs/Dialog[@Name='Dialog007']", Params
'taskDialog.CreateFromXML LoadDialogs, "Dialogs/Dialog[@Name='Dialog007']", Array("SampleFileName.txt", "Text Document", "www.Codejock.com")
taskDialog.ShowDialog

Function LoadDialogs() As String
    On Error Resume Next

    Dim Dialogs As String
    Dim FreeF As Integer, sFile As String, sLayout As String


    'XML string will be loaded from the file.
    sFile = App.Path & "\Dialogs.xml"
    FreeF = FreeFile
    Dialogs = Space(FileLen(sFile))
    Open sFile For Binary As #FreeF
    Get #FreeF, , Dialogs
    Close #FreeF
    On Error Goto 0

    LoadDialogs = Dialogs

End Function
