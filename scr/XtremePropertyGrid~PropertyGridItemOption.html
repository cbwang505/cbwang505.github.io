Xtreme Property Grid ActiveX Control v15.3

PropertyGridItemOption Object

Description

Represents an option button or check box.

For a list of all members defined in this module, see PropertyGridItemOption members.

Object Model














Remarks

The PropertyGridItemOption can be displayed as a check box or a option button.  Set CheckBoxStyle to True to use a check box style.

Example

[ItemOption Example (Visual Basic)] 

    Dim ItemOption As PropertyGridItemOption
    Set ItemOption = Category.AddChildItem(PropertyItemOption, "Options", 1)
    ItemOption.Constraints.Add "Option 1", 1
    ItemOption.Constraints.Add "Option 2", 2

    Set ItemOption = Category.AddChildItem(PropertyItemOption, "Options", 1 + 2)
    ItemOption.CheckBoxStyle = True
    ItemOption.Constraints.Add "Option 1", 1
    ItemOption.Constraints.Add "Option 2", 2
    'ItemOption.Constraints.Add "Option 3", 4
    'ItemOption.Constraints.Add "Option 4", 8
