Xtreme Docking Pane ActiveX Control v15.3

CreatePane Method

Description

Call this member function to create a docking pane.

Syntax

Public Function CreatePane( _
   ByVal Id As Long, _
   ByVal cx As Long, _
   ByVal cy As Long, _
   ByVal Direction As DockingDirection, _
   Optional ByVal Neighbour As Variant _
) As Pane

Parameters

Id

Unique identifier of the pane

cx

The initial width of the pane.

cy

The initial height of the pane.

Direction
ValueDescription
DockLeftOfDocks a pane to the left of an existing pane.
DockRightOfDocks a pane to the right of an existing pane
DockTopOfDocks a pane to the top of an existing pane
DockBottomOfDocks a pane to the bottom of an existing pane

Docking direction. It can be one of the following:

  • dockTopof - To dock pane to the top of the neighbor
  • dockLeftof - To dock pane to the left of the neighbor
  • dockRightof - To dock pane to the right of the neighbor
  • dockBottomof - To dock pane to the bottom of the neighbor
Neighbour

Pane's Neighbor. It can be NULL to use the frame as a neighbor.

Return Type

A Pane object.

Remarks

When creating a pane, you specify the docking direction, which specifies the direction the pane will be docked in relation to another pane or one of the client boarders.
