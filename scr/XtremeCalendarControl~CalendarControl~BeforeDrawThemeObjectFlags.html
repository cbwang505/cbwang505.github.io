Xtreme Calendar ActiveX Control v15.3

BeforeDrawThemeObjectFlags Property

Description

Interface used to get and set which calendar objects will be custom drawn when the Office 2007 theme is used.

Property type

Read-only property

Syntax (Visual Basic)

Public Property BeforeDrawThemeObjectFlags() As CalendarFlagsSet

Remarks

BeforeDrawThemeObjectFlags returns a CalendarFlagsSet object which is used as an interface to get and set which Office 2007 calendar objects will be custom drawn.  The interface includes the SetFlag, IsFlagSet and ResetFlag methods which allow you to set, get, and reset the CalendarBeforeDrawThemeObject flags which specify for which Office 2007 calendar objects the BeforeDrawThemeObject event will get fired.

 

The SetFlag method uses any combination of flags from the CalendarBeforeDrawThemeObject enumeration to specify for which Office 2007 calendar objects the BeforeDrawThemeObject event will get fired.  The BeforeDrawThemeObject event occurs when each Office 2007 theme component of the calendar is about to be drawn if some Flags from the CalendarBeforeDrawThemeObject enumeration have been set using the SetFlag method.  The SetFlag method can be used to set any combination of the CalendarBeforeDrawThemeObject (only use with Office 2007 theme) flags.  BeforeDrawThemeObject will only get fired when drawing components that are specified using the SetFlag method. 

 

If you would like the BeforeDrawThemeObject event to get fired for all flags in the CalendarBeforeDrawThemeObject enumeration then you can simply set BeforeDrawThemeObjectFlags to -1.  This will cause the BeforeDrawThemeObject event to fire before each part of the calendar is drawn.

 

This allows all of the parts of the calendar to easily be customized.
