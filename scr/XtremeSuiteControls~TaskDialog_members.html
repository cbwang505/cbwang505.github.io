TaskDialog Control Members

Public Methods

 AboutBoxDisplays the about box for the control
 AddButtonAdds a user defined button to the task dialog's command area.
 AddInputBoxAdds a user input box to the task dialog's command area.
 AddRadioButtonAdds a user defined radio button to the task dialog's command area.
 ClickButtonSimulate the action of a button click in the task dialog.
 ClickRadioButtonSimulate the action of a radio button click in the task dialog.
 CreateFromXMLCreates a dialog from a XML resource file.
 EnableButtonEnables or disables a button in the task dialog.
 EnableRadioButtonEnables or disables a radio button in the task dialog.
 ResetResets the configuration attributes for the task dialog to the default state.
 SetProgressBarRangeSets the minimum and maximum values for the progress bar.
 ShowDialogDisplays the task dialog and return the selected button when done.
 StartProgressBarMarqueeSpecifies whether the progress bar should be displayed in marquee mode or not with the ability to determine marquee speed as well.

Public Properties

 AllowCancellationGets or sets whether to allow dialog cancellation for the task dialog.
 CollapsedControlTextGets or sets the collapsed control text for the task dialog.  This property is ignored unless ExpandedInformationText has been set.
 CommonButtonsSpecifies the "commmon" push buttons displayed in the task dialog.  If no common buttons are specified and no custom buttons are specified using the AddButton method, the task dialog will contain the "OK" button by default.
 ContentTextGets or sets the content text for the task dialog.
 DefaultButtonGets or sets the default button ID for the task dialog.
 DefaultRadioButtonGets or sets the default radio button ID for the task dialog.
 DialogWidthGets or sets the width of the task dialog's client area.
 EnableCallbackTimerEnables or disables the callback timer for this task dialog.
 EnableCommandLinksSpecifies whether the buttons specified with AddButton should be displayed as command links (using a standard task dialog glyph) instead of push buttons.  This property is ignored if AddButton has not been called.
 EnableHyperlinksEnables hyperlink displaying and processing for the strings specified for the ContentText, ExpandedInformationText and FooterText properties.
 EnableMarkupAllows the use of a XAML snippet to be used for the Caption of this control.
 ExpandedByDefaultGets or sets whether to display the text specified by the ExpandedInformationText property when the task dialog is initially displayed.  This property is ignored unless ExpandedInformationText has been set.
 ExpandedControlTextGets or sets the expanded control text for the task dialog.  This property is ignored unless ExpandedInformationText has been set.
 ExpandedInformationTextGets or sets the expanded information text for the task dialog.
 ExpandFooterAreaGets or sets whether to display expanded information at the bottom of the task dialog's footer.  This function is ignored if the ExpandedInformationText property has not been set.
 FooterIconGets or sets the icon for the Footer area of the task dialog.
 FooterIconHandleGets or sets the handle for a custom icon for the Footer area of the task dialog.
 FooterTextGets or sets the footer text for the task dialog.
 InputBoxTextGets or sets the text entered by user in input box field.  Or sets the default value for input box field.
 MainIconGets or sets the icon for the Main Instruction area of the task dialog.
 MainIconHandleGets or sets the handle for a custom icon for the Main Instruction area of the task dialog.
 MainInstructionTextGets or sets the main instruction text for the task dialog.
 MarkupContextConnects events in XML Markup with actual methods, functions and events in your code.
 MessageBoxStyleSpecifies whether to show the dialog using the Windowx XP or Windows Vista style even if the standard window theme or no manifest is used.
 ParentHwndGets or sets the handle to the parent window for the task dialog.
 ProgressBarPosGets or seta the current position for the progress bar.
 ProgressBarStyleGets or sets the style of the progress bar.  This is used to display the progress bar and specifies whether the progress bar should be displayed in marquee mode or not.
 RelativePositionSpecifies whether to position the task dialog relative to the parent window.
 RtlLayoutEnables or disables right to left text display for the task dialog
 ShowCommandLinkIconsSpecifies whether the command links should display an incon next to the button text.
 UseComCtl32Gets or sets whether to use the ComCtl32.dll (Windows Vista) version of Task Dialog.
 VerificationTextGets or sets the label for the verification checkbox to be displayed in the task dialog.
 VerifyCheckStateGets or sets the checked state for the verification checkbox when the dialog is initially displayed.  This property is ignored if the VerificationText property has not been set.
 WindowTitleGets or sets the window title for the task dialog.

Events

 ButtonClickedOccurs when a user selects a button or command link in the task dialog.
 ConstructedOccurs once the task dialog has been created and before it is displayed.
 ExpandButtonClickedOccurs when the user clicks on the expand in the Task Dialog.
 HyperlinkClickedOccurs when a user clicks on a hyperlink in the Task Dialog's content.
 InputBoxChangedOccurs when the InputBoxText has changed in the input box.
 RadioButtonClickedOccurs when a user selects a radio button in the task dialog.
 TimerThis event is sent by the Task Dialog approximately every 200 milliseconds when the EnableCallbackTimer property has been set.
 VerificationClickedOccurs when the user clicks on the task dialog's verification check box.
