CalendarEventLabels Collection

Description

Represents a collection of CalendarEventLabel objects.

For a list of all members defined in this module, see CalendarEventLabels members.

Object Model

Remarks

The LabelList property is a CalendarEventLabels collection used to store CalendarEventLabel objects that can be used as labels for events.

 

The table below lists all the default label values and colors that are added to the LabelList when using InitDefaultValues:

Default Background Color Default Name Default LabelID
RGB(255,255,255) None 0
RGB(255,148,132) Important 1
RGB(132,156,231) Business 2
RGB(165,222, 99) Personal 3
RGB(231,231,214) Vacation 4
RGB(255,181,115) Must Attend 5
RGB(132,239,247) Travel Required 6
RGB(214,206,132) Needs Preparation 7
RGB(198,165,247) Birthday 8
RGB(165,206,198) Anniversary 9
RGB(255,231,115) Phone Call 10

 

Calling the InitDefaultValues method will populate the LabelList collection with the values in the table above.

 

You can Add or Remove labels to the LabelList to be used for events.  This allows you to set a custom color for the label.
