ShortcutBar Overview

Xtreme Shortuct Bar ActiveX Control v15.3

ShortcutBar Control Members

Public Methods

 AboutBoxDisplays the About Box for the control
 AddImageListAdds icons from an image list that are used to display ShortcutBar items
 AddItemAdds a new shortcut to the ShortcutBar.
 FindItemFinds a shortcut in the ShortcutBar.
 HitTestThe HitTest method determines if a ShortcutBarItem is at a specific X and Y coordinate.
 RemoveAllItemsRemoves all shortcuts from the ShortcutBar.
 RemoveItemRemoves the specified shortcut from the ShortcutBar.

Public Properties

 AllowFreeResizeAllows the items in the shortcut bar to be expanded by pixel instead of by item.
 AllowGripperResizeGets\sets whether the resize gripper will be enabled/disabled.
 AllowMinimizeSpecifies whether the ShortcutBar will minimize to a "Navigation Pane" when the width of the bar becomes real small.
 ClientHeightReturns the current height of the client area.
 ClientLeftReturns the distance between the internal left edge of the ShortcutBar container and the left edge of the ShortcutBar client area.
 ClientPaneVisibleShows or hides the client panes associated with the shortcuts.
 ClientTopReturns the distance between the internal top edge of the ShortcutBar container and the top edge of the ShortcutBar client area.
 ClientWidthReturns the current width of the client area.
 ColorSetContains all the colors in the ShortcutBar.
 EnableAnimationEnables or disables the animations used in the Shortcut Bar (i.e. when expanding the shortcut list).
 EnabledGets/Sets a boolean value determining whether the control is enabled
 ExpandedLinesCountGets/Sets the number of lines (shortcuts) in the expanded shortcut list.
 HighlightedThe ShortcutBarItem that the mouse is currently positioned over in the ShortcutBar control.
 hWndThe Handle to the object that the ShortcutBar is bound to
 IconsGet\Sets the icons used in the ShortcutBar.
 ItemThe Item property is a collection of ShortcutBarItems that hold all of the items added to the Shortcut Bar.
 ItemCountThe ItemCount property returns the total number of items in the Shortcut Bar.
 ItemHeightGets or sets the height of items in the shortcut bar.
 ItemWidthGets or sets the width of items in the shortcut bar.
 MinimizedWidthGets\sets the width at which the shortcutbar will become minimized and display as a navigation pane.
 MinimumClientHeightThe MinimumClientHeight property specifies the minimum height of the ShortcutBar client area.
 PaintManagerProvides access to various shortcut bar painting settings.
 PopupWidthSpecifies the width of the popup shortcurbar when the bar is minimized.
 RightToLeftGets or sets a value indicating whether the ShortcutBar elements are aligned to support locales using right-to-left fonts.
 SelectedGets/sets the currently selected shortcut in the ShortcutBar control.
 ShowActiveItemOnTopSpecifies whether the currently active shortcut item will be displayed above the client area.
 ShowExpandButtonShows or hides the expand button.
 SingleSelectionThe SingleSelection property specifies whether only one item can be selected in the ShorcutBar at a time.
 ToolTipContextThe ToolTipContext object represents the tool tip window that is used to display tool tips.
 VisualThemeGets/Sets the visual theme of the ShortcutBar control

Events

 ClientSizeChangedThe ClientSizeChanged event occurs when the client area of the ShortcutBar is resized.
 ExpandButtonDownThe ExpandButtonDown event occurs when the expand button in the Shortcut Bar is pressed.
 HighlightedChangedOccurs when the mouse cursor is positioned over a ShortcutBarItem and the item becomes Highlighted.
 MouseUpThe MouseUp event occurs when a button on the mouse has been released anywhere on the Shortcut Bar.
 SelectedChangedOccurs each time a new shortcut in the ShortcutBar is selected.
 SelectedChangingOccurs when the selected item is in the process of changing, but has not changed.
