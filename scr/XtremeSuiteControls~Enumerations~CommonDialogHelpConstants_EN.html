Xtreme Suite Controls ActiveX Control v15.3

CommonDialogHelpConstants Enumeration

Description

Help Dialog Constants.

Members

MemberDescription
xtpHelpCommandHelpExecutes a Help macro.
xtpHelpContentsDisplays the Help contents topic as defined by the Contents option in the [OPTION] section of the .hpj file. See Remarks below for information on Help files created with Microsoft Help Workshop 4.0X.
xtpHelpContextDisplays Help for a particular context. When using this setting, you must also specify a context using the HelpContext property.
xtpHelpContextPopupDisplays in a pop-up window a particular Help topic identified by a context number defined in the [MAP] section of the .hpj file.
xtpHelpForceFileEnsures WinHelp displays the correct Help file. If the correct Help file is currently displayed, no action occurs. If the incorrect Help file is displayed, WinHelp opens the correct file.
xtpHelpHelpOnHelpDisplays Help for using the Help application itself.
xtpHelpIndexDisplays the index of the specified Help file. An application should use this value only for a Help file with a single index.
xtpHelpKeyDisplays Help for a particular keyword. When using this setting, you must also specify a keyword using the HelpKey property.
xtpHelpPartialKeyDisplays the topic found in the keyword list that matches the keyword passed in the dwData parameter if there is one exact match. If more than one match exists, the Search dialog box with the topics found listed in the Go To list box is displayed. If no match exists, the Search dialog box is displayed. To bring up the Search dialog box without passing a keyword, use a long pointer to an empty string.
xtpHelpQuitNotifies the Help application that the specified Help file is no longer in use.
xtpHelpSetContentsDetermines which contents topic is displayed when a user presses the F1 key.
xtpHelpSetIndexSets the context specified by the HelpContext property as the current index for the Help file specified by the HelpFile property. This index remains current until the user accesses a different Help file. Use this value only for Help files with more than one index.

Remarks

The HelpCommand property uses values from the CommonDialogHelpConstants enumeration to specify the type of help.
