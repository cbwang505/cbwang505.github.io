Public Properties

 AlignmentReturns or sets a value that determines the alignment of the UpDown control with its buddy control.
 AppearanceGets or sets the paint style of the control.
 AutoBuddySets or returns a value that determines whether the UpDown control automatically uses a control as its buddy control, based on its tab order.
 BuddyControlSets or returns the control used as the buddy control.
 BuddyPropertySets or returns the property used to synchronize the UpDown control with its buddy control.
 ControlToolTipThe ControlToolTip property gets or sets the tool tip that is displayed when the mouse cursor is positioned over this item.
 EnabledGets or set whether this control is enabled.
 IncrementSets or returns a value that determines the amount by which the Value property changes when the UpDown control's buttons are clicked.
 MaxSets or returns the maximum value of the scroll range for the UpDown control.
 MinSets or returns the minimum value of the scroll range for the UpDown control.
 OrientationSets or returns the placement of the arrow buttons on the UpDown control. This property is read-only at run time.
 SyncBuddySets or returns a value that determines whether the UpDown control synchronizes the Value property with a property in the buddy control.
 ToolTipContextThe ToolTipContext object represents the tool tip window that is used to display tool tips.
 UseVisualStyleDetermines if the control is drawn using visual styles\SkinFramework, if supported.
 ValueSets or returns the current position of the scroll value.

Events

 ChangeOccurs when the Value property is changed.
 DownClickOccurs when the down or left arrow button is clicked.
 UpClickOccurs when the up or right arrow button is clicked.
