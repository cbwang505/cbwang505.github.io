ListView Control

Description

The ListView control displays items using one of four different views. You can arrange items into columns with or without column headings as well as display accompanying icons and text.

For a list of all members defined in this module, see ListView members.

Object Model











Remarks

With a ListView control, you can organize list entries, called ListViewItem objects, into one of four different views: Large Icons (xtpListViewIcon), Small Icons (xtpListViewSmallIcon), List (xtpListViewIcon) and Report (xtpListViewReport) views.

The View property determines which view the control uses to display the items in the list. You can also control whether the labels associated with items in the list wrap to more than one line using the LabelWrap property. In addition, you can manage how items in the list are sorted and how selected items appear.

The ListView control contains ListViewItem and ListViewColumnHeader objects. A ListViewItem object defines the various characteristics of items in the ListView control, such as: A brief description of the item, Icons that may appear with the item and Additional pieces of text associated with a ListViewItem object that you can display in Report view.

You can choose to display column headings in the ListView control using the HideColumnHeaders property. Use the Add method to add a ListViewColumnHeader object to the ColumnHeaders collection.
