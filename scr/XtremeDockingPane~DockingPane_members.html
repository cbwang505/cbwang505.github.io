DockingPane Overview

Xtreme Docking Pane ActiveX Control v15.3

DockingPane Control Members

Public Methods

 AboutBoxDisplays a dialog box that specifies the current version of the control.
 AddIconHandleAdds an icon to be used in the docking pane tabs.
 AttachPaneCall this member to attach a pane to another pane creating a group of panes.
 AttachToWindowSpecifies the control that will contain the Docking Panes.
 CloseAllCloses all panes
 CreateLayoutThe CreateLayout method creates a DockingPaneLayout object to store a docking pane state/layout.  The DockingPaneLayout object created is used with the GetLayout and SetLayout methods to load and set docking pane layouts.
 CreatePaneCall this member function to create a docking pane.
 DestroyAllDestroys all panes
 DestroyPaneDestroys the specified pane
 DockPaneMoves\docks an existing pane to another side of an application or pane.
 DockSidePaneDocks a pane to an inside edge of the client area when side docking is enabled.
 EnableKeyboardNavigateSpecifies the keyboard keys that can be used to navigate the docking panes.
 FindPane

Finds and returns the specified docking pane, searching by pane Id.

 FloatPaneFloats the specified pane and resizes the pane with the given dimensions.
 GetClientRectRetrieves the dimensions of the client rectangle, that is, the area on the form which does not have any docking panes.  This is needed so you know where the docking panes can be drawn.
 GetLayoutThe GetLayout method retrieves the currently used docking pane layout.
 HidePaneCall this member to hide a pane.
 LoadStateLoads the pane layout from the registry.
 LoadStateFromStringLoads the pane layout from a string.
 NormalizeSplittersUsed to remember the location of splitters.
 RecalcLayoutRecalculates the entire layout of the docking panes on the form.
 RedrawPanesRedraws the panes
 SaveStateSaves the pane layout to the registry.
 SaveStateToStringSaves the pane layout to a string
 SetCommandBarsSets a CommandBars object to associate with this Docking Pane object.
 SetLayoutThe SetLayout method sets the layout of the docking panes.  The SetLayout method set a state\layout saved by the GetLayout method.
 SetMDIClientUsed in .NET to set handle to MDI client window.
 ShowPaneCall this member to show (activate) a pane by its identifier.
 UpdatePanesUpdates the state of the panes.

Public Properties

 ActivePaneThe ActivePane property holds a reference to the currently active Pane.
 IconsA collection of icons stored in an ImageManager control. This allows you to use images stored in the CommandBars ImageManager control in your docking pane tabs.  The images from the ImageManager are displayed in the docking pane tabs
 ImageListSets/Gets the imagelist associated with DockingPanes.
 LayoutUsed to store a docking pane layout. 
 OptionsGets/Sets the options of the docking pane
 PaintManagerThe PaintManager object is used to hold various pane painting properties.
 PanelPaintManagerGets the panel PaintManager properties.  The panel paint manager is used to customize the appearance of your docking pane auto hide panel.  The auto hide panel is displayed when you hide your docking panes.
 PanesThe DockingPaneManager's collection of panes.
 PanesCountGets the count of the total number of panes present.
 ScaleModeUsed to change the Docking Pane's measurement units/coordinate system
 TabPaintManagerGets the tab PaintManager properties.  The tab paint manager is used to customize the appearance of your docking pane tabs when the docking pane is NOT hidden.
 ToolTipContextThe ToolTipContext object represents the tool tip window that is used to display tool tips.
 VisualThemeGets/Sets the visual theme which is used by the docking panes.

Events

 ActionOccurs when a docking pane is performing an action.
 AttachPaneOccurs once when a pane is about to be displayed.
 GetClientBordersWidthOccurs when the docking pane layout changes.
 PanePopupMenuOccurs when the pane caption Menu Button is pressed.
 RClickThe RClick event occurs when the user right-clicks anywhere on the pane caption or on the tab button of a pane.
 ResizeOccurs when the docking pane layout changes.
 ResizeClientOccurs when the docking pane layout changes.
