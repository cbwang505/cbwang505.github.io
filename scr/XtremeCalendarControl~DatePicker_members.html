DatePicker Overview

Xtreme Calendar ActiveX Control v15.3

DatePicker Control Members

Public Methods

 AttachToCalendarAssociates a DatePicker Control with a Calendar Control.
 ClearSelectionClears all the currently selected days in the date picker.
 DeselectDeselects a specific day in the date picker.
 DesiredVisibleAndFullRangeAllow scrolling between FirstDay and LastDay and trys to show initial visible range between FirstVisibleDay and LastVisibleDay.
 DetachFromCalendarUn-associates a DatePicker Control with a Calendar Control.
 EnsureVisibleCall this member function to ensure that a specific Date is visible in the DatePicker.
 EnsureVisibleFocusEnsures that the focused item is visible.
 EnsureVisibleSelectionEnsures that the current selection range is visible.
 GetMinReqRectThis member function retrieves the minimum size required to show a specified number of full month items in a date picker control.
 HitTestDetermines which Date in the date picker is affected by the mouse pointer.
 HitTestExDetermines which date portion of the date picker control, if any, is at a specified position.
 RedrawControlCall this method to redraw the DatePicker Control.
 ResyncCalendarRe-syncs the data in the date picker with the calendar's data.
 SelectSelects a specific day in the date picker control.
 SelectRangeThis member function sets the selection for a date picker control to a given date range.  This will "highlight" the dates in the given range.
 SetRangeThis member function sets the minimum and maximum allowable dates for a date picker control.
 ShowModalDisplays a Date Picker control as a modal dialog with a specific number of month rows and columns.
 ShowModalExDisplays a Date Picker control as a modal dialog.  Allows you to specify the top left corner of the date picker and the width\height.

Public Properties

 AllowNoncontinuousSelectionSpecifies whether the date picker control allows non-continuous days selection or not.
 AskDayMetricsSpecifies whether to fire the DayMetric event.
 AutoSizeRowColGets or sets the AutoSize mode.
 BoldDaysIdleStepTime_msGets/sets the time between idle steps to update the bold state of days in the date picker.
 BoldDaysOnIdleEnables or disables retrieving the bold state of days in the date picker by using a timer.
 BoldDaysPerIdleStepGets/sets the total number of days in the date picker whose bold status will be updated in a single Idle Step.
 BoldDaysWithEventsGets\sets wether dates that correspond to days in the Calendar that contain events will be displayed in bold font.
 BorderStyleGets\sets the border style around the Date Picker control.
 ColumnCountGets\sets the number of columns in the months grid.
 EnabledEnables\disables the date picker control.
 FirstDayOfWeekGets\sets the day of the week (Monday - Sunday) that is displayed in the leftmost column of the calendar.
 FirstVisibleDayGets the first visible date in the date picker.
 FirstWeekOfYearDaysThe FirstWeekOfYearDays property contains the current required number of year days that occur in the first week of the year.
 HighlightTodayEnables or disables highlighting of the "Today" cell.
 hWndReturns the windows Handle to the Date Picker Control.
 LastVisibleDayGets the last visible date in the date picker.
 MaxSelectionCountGets\sets the maximum number of days that can be selected in a date picker control.
 MonthDeltaGets or sets the scroll rate for the date picker control.
 MultiSelectionModeEnables/disables the multiple selection mode for the control.
 PaintManagerThis paint manager handles most of the drawing activities for the DatePicker control.
 RightToLeftGets or sets a value indicating whether the Date Picker elements are aligned to support locales using right-to-left fonts.
 RowCountGets\sets the number of rows in the months grid.
 SelectionRetrieves date information representing the upper and lower limits of the date range currently selected by the user.
 ShowNoneButtonHides or shows the Date Picker Control's "None" button.
 ShowNonMonthDaysShows or hides non-month days.
 ShowTodayButtonHides or shows the Date Picker Control's "Today" button.
 ShowWeekNumbersShows or hides the week numbers for month items displayed in the Date Picker control.
 TextNoneButtonGets\sets the text that is displayed in the "None" button.
 TextTodayButtonGets\sets the text that is displayed in the "Today" button.
 VisualThemeGets/Sets the visual theme which is used to display date picker.
 YearsTriangleHides or displays the year triangle button in the date picker caption.

Events

 DayMetricsOccurs before the each date\day is drawn in the Date Picker.  This allows the metrics to be modified before the day is displayed.
 KeyDownOccurs when a key is pressed and the date picker control has focus.
 MonthChangedOccurs when the months currently in view have changed.
 MouseMoveThe MouseMove event occurs when the mouse cursor is moved anywhere over the Date Picker Control.
 SelectionChangedOccurs when the selection has changed in the Date Picker Control.
