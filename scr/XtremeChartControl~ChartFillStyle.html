Xtreme Chart Pro ActiveX Control v15.3

ChartFillStyle Object

Description

Represents the styles used to fill\shade the background of various elements of the chart (strips, interlacing, background, markers, labels, legend, etc) .

For a list of all members defined in this module, see ChartFillStyle members.

Remarks

The Fill Style allows the Fill Mode, Gradient Angle and Hatch Style to be set.  FillMode applies to the background (columns, interlacing, stripes, markers, labels, legend, etc) of many of the various elements in the chart.

 

Fill Mode must be set to xtpChartFillGradient to use a gradient or xtpChartFillHatch to use a hatch style.  Note both gradient and hatch can't be used at the same time, you can use one or the other, but not both a the same time.  Fill Mode can also be set to solid or empty and no gradient or hatch will be used.

 

The background fill of 2D charts can be set using the BackgroundFillStyle property.  The gradients and hatch styles can be applied to any 2D axis background.  The colors used for the background gradient can be set using the BackgroundColor and BackgroundColor2 properties.

 

Both the X and Y Axis have an InterlacedFillStyle property that allows the fill style to be set for interlaced "columns" along that axis, if Interlacing is turned on.  The colors used for the interlaced gradient can be set using the InterlacedColor and InterlacedColor2 properties.

 

The Axis Strip FillStyle property allows the fill mode of axis "strips" to be set.

Example

'This sample will set the fill mode for all strips and interlacing for all axis

        Dim FillModeIndex As ChartFillMode

        FillModeIndex = xtpChartFillSolid

        If ActiveChartControl.Content.Series.Count > 0 Then
            Dim xSeries As ChartSeries
            For Each xSeries In ActiveChartControl.Content.Series
                Set Diagram = xSeries.Diagram
                Dim xStrip
               
                Diagram.AxisX.InterlacedFillStyle.FillMode = FillModeIndex
                For Each xStrip In Diagram.AxisX.Strips
                    xStrip.FillStyle.FillMode = FillModeIndex
                Next
                Diagram.AxisY.InterlacedFillStyle.FillMode = FillModeIndex
                For Each xStrip In Diagram.AxisY.Strips
                    xStrip.FillStyle.FillMode = FillModeIndex
                Next
                Diagram.AxisX.SecondaryAxisX.FillMode = FillModeIndex
                For Each xStrip In Diagram.SecondaryAxisX.Strips
                    xStrip.FillStyle.FillMode = FillModeIndex
                Next
                Diagram.SecondaryAxisY.InterlacedFillStyle.FillMode = FillModeIndex
                For Each xStrip In Diagram.SecondaryAxisY.Strips
                    xStrip.FillStyle.FillMode = FillModeIndex
                Next
            Next
        End If

 

'This sample will set the gradient for all strips and interlacing for all axis

        Dim GradientAngleIndex As ChartGradientAngle
        GradientAngleIndex = xtpChartGradientLeftToRight
       
        If ActiveChartControl.Content.Series.Count > 0 Then
            Dim xSeries As ChartSeries
            For Each xSeries In ActiveChartControl.Content.Series
                Set Diagram = xSeries.Diagram
                Dim xStrip
                Diagram.AxisX.InterlacedFillStyle.GradientAngle = GradientAngleIndex

                    xStrip.FillStyle.GradientAngle = GradientAngleIndex
                Next
                Diagram.AxisY.InterlacedFillStyle.GradientAngle = GradientAngleIndex
                For Each xStrip In Diagram.AxisY.Strips
                    xStrip.FillStyle.GradientAngle = GradientAngleIndex
                Next
                Diagram.SecondaryAxisX.InterlacedFillStyle.GradientAngle = GradientAngleIndex
                For Each xStrip In Diagram.SecondaryAxisX.Strips
                    xStrip.FillStyle.GradientAngle = GradientAngleIndex
                Next
                Diagram.SecondaryAxisY.InterlacedFillStyle.GradientAngle = GradientAngleIndex
                For Each xStrip In Diagram.SecondaryAxisY.Strips
                    xStrip.FillStyle.GradientAngle = GradientAngleIndex
                Next
            Next
        End If

 

'This sample will set the hatch style for all strips and interlacing for all axis

        HatchIndex = xtpChartHatchStyle25Percent
