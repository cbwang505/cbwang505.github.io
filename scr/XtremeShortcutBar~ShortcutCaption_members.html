ShortcutCaption Overview

Xtreme Shortuct Bar ActiveX Control v15.3

ShortcutCaption Control Members

Public Methods

 AboutBoxDisplays the About Box for the control

Public Properties

 AlignmentGets/Sets the alignment of the text within the ShortcutCaption.
 CaptionGets/Sets the caption of the ShortcutCaption.  This is the text that will be displayed in the ShortcutCaption.
 ExpandableSpecifies whether the caption will have an expand button.
 ExpandedSpecifies whether the caption is expanded (expand arrow orientation).
 FontGets/Sets the font used to display text in the ShortcutCaption.
 ForeColorGets\sets the color of the text displayed in the Shortcut Caption.
 GradientColorDarkThe GradientColorDark property is the "dark" color in the color gradient of a Shortcut Caption.
 GradientColorLightThe GradientColorLight property is the "light" color in the color gradient of a Shortcut Caption.
 GradientHorizontalThe GradientHorizontal property is used to specify the direction of the color gradient in the Shortcut Caption.
 IconGets\sets the Id of an Icon from the ImageManager to use for this ShortcutCaption.
 RightToLeftReadingAllows Right-To-Left (RTL) reading order without changing the entire layout of the control (links, icon order, etc).
 SubItemCaptionGets/Sets whether the ShortcutCaption bar is a sub caption.  Sub captions are displayed in another color and are usually placed underneath normal ShortcutCaption's.
 VisualThemeGets/Sets the visual theme of the ShortcutCaption.

Events

 ExpandButtonClickedOccurs when the expand button is clicked on the ShortcutCaption control.
 MouseDownOccurs when a button on the mouse is pressed anywhere on the ShortcutCaption control.
 MouseUpOccurs when a button on the mouse has been released anywhere on the ShortcutCaption control.
