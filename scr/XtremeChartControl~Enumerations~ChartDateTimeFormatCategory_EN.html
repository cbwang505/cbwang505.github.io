ChartDateTimeFormatCategory Enumeration

Description

Defines the various DateTime formats available to format labels in the chart.

Members

MemberDescription
xtpChartDateTimeGeneralStandard formatting.
xtpChartDateTimeLongDateLong date format, i.e. Friday, January 01, 2012
xtpChartDateTimeShortDateShort date format, i.e. 1/1/2012
xtpChartDateTimeLongTimeLong time format, i.e. 12:00:00 AM
xtpChartDateTimeShortTimeShort time format, i.e. 12:00 AM
xtpChartDateTimeMonthAndDayMonth and Day format, i.e. January 26
xtpChartDateTimeMonthAndYearMonth and Year format, i.e. January 2012
xtpChartDateTimeCustomUses a custom format specified in DateTimeFormat.

Remarks

DateTimeCategory uses values from the ChartDateTimeFormatCategory enumeration to tell the chart how to handle the formatting of DateTime values in labels.  The category must be specified so the correct formatting is applied to the value.  This is used when the ArgumentScaleType is set to xtpChartScaleDateTime.  The formatting can be set to General, Short Date, Long Date, Short Time, Long Time, Month and Day, Month and Year or Custom.  For custom, the format uses DateTimeFormat.
